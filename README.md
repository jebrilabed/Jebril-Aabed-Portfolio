# Jebril Aabed | Modern Portfolio 🚀

A high-performance, visually stunning personal portfolio built with **Next.js 15**, **React 19**, **Tailwind CSS**, and **Framer Motion**. This project showcases my journey as a Frontend Developer, featuring interactive project displays, a responsive design system, and a smooth user experience.

![Portfolio Preview](https://ebril-Aabed-Portfolio.vercel.app/)

## ✨ Features

- **Modern Tech Stack**: Built with Next.js 15 and React 19 for optimal performance.
- **Dynamic Animations**: Powered by Framer Motion for smooth transitions and interactive UI elements.
- **Fully Responsive**: Mobile-first design using Tailwind CSS.
- **Project Showcase**: Highlighted projects with descriptions, tech tags, and live demos.
- **Contact Integration**: Integrated contact form with API route handling.
- **SEO Optimized**: Optimized metadata and semantic HTML for better search engine ranking.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/jebrilabed/Jebril-Aabed-Portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   Create a `.env.local` file in the root directory and add your configuration (e.g., email services).

4. **Run the development server**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to see the result.
